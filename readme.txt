== Description ==

Add Multiple Users From Network is based on the fine work by happynuclear http://addmultipleusers.happynuclear.com.

== Installation ==

Installing and activating this plugin adds an additional menu option to your menu in the WordPress administration
section named 'AMU'.

1. Upload 'add-multiple-users-from-network' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the Users -> Add from Network tab in the left menu of your admin menu to access plugin functions
